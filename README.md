## 前序准备
### 运行前准备
请确保本地已经安装node（8.0及以上版本）

## 环境搭建
```bash
# 克隆版本库
git clone https://github.com/tech-lu/frontend-mobile.git
# 安装依赖包
cd frontend-mobile
npm install

# 以开发模式运行
npm run serve
```

## 发布
```bash
npm run build
```
